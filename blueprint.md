# Visor de Actas Electorales

## Descripción

Esta aplicación permite a los usuarios ver y buscar actas electorales escaneadas y procesadas por un sistema de IA. La aplicación proporciona una interfaz de usuario para que los usuarios autorizados puedan ver la información de las actas electorales y buscar por número de mesa.

## Características

- **Autenticación de usuarios:** Sistema de login para usuarios autorizados.
- **Dashboard:** Muestra un resumen de las actas procesadas.
- **Búsqueda de actas:** Permite buscar actas por número de mesa.
- **Visualización de actas:** Muestra la información del acta y la imagen escaneada.

## Diseño y Estilo

- **Framework:** Next.js con App Router.
- **Estilos:** Tailwind CSS para un diseño moderno y responsivo.
- **Componentes:**
    - **Header:** Barra de navegación superior.
    - **Sidebar:** Menú lateral para la navegación principal.
- **Paleta de colores:**
    - **Fondo:** Gris claro (`bg-gray-100`).
    - **Primario:** Azul (`bg-blue-600`).
    - **Peligro:** Rojo (`bg-red-600`).
